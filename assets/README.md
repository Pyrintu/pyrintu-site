# assets/ — logo placeholders and instructions

This folder should contain your logo files. Upload the PNGs you provided into this path:

- assets/logo-horizontal.png  (header horizontal lockup)
- assets/logo-square.png      (square mark used in footer and social)
- assets/logo-mark.png        (optional small mark)

If you have SVGs, prefer those and replace the PNG files. After uploading, verify the site on GitHub Pages.
